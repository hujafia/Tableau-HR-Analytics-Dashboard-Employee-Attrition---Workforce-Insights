# 📊 HR Analytics Dashboard  
## Employee Attrition Analysis using Tableau

---

## 📌 Project Overview

This project presents an **interactive HR Analytics Dashboard** built using **Tableau** to analyze employee attrition, workforce demographics, and job satisfaction patterns.

The goal of this dashboard is to help organizations **identify the key drivers behind employee attrition** and enable HR teams to make **data-driven workforce decisions**.

By transforming HR data into meaningful visual insights, this dashboard highlights **which employee groups are more likely to leave and where organizations should focus their retention strategies.**

## 🎯 Project Objective

Employee attrition is one of the biggest challenges faced by many organizations.

This project analyzes HR data to answer important business questions such as:

- Which departments experience the highest attrition?
- Are certain age groups more likely to leave the company?
- Does job satisfaction impact employee retention?
- Are there gender differences in attrition trends?
- Which education fields have higher employee turnover?

The dashboard provides a **clear and interactive overview of workforce trends** to support better HR decision-making.

---

## 📈 Key Metrics (KPIs)

The dashboard highlights several important HR performance indicators:

- **Total Employees**
- **Attrition Count**
- **Attrition Rate**
- **Average Job Satisfaction Score**
- **Employee Distribution by Age Group**

These KPIs provide a **quick overview of the overall workforce health of the organization.**

---

## 📊 Dashboard Insights

### 🔹 Attrition by Gender & Age Group
This visualization highlights how attrition varies across **different age groups and genders**, helping HR teams identify which demographic segments require more attention.

### 🔹 Department-wise Attrition
This section identifies departments with the **highest employee turnover**, allowing organizations to detect potential management or workload issues.

### 🔹 Education Field-wise Attrition
The dashboard explores attrition patterns based on **educational background**, revealing which educational fields experience higher employee turnover.

### 🔹 Gender-based Attrition
Provides insight into **gender differences in employee attrition**, helping HR teams evaluate diversity and retention strategies.

### 🔹 Job Satisfaction Analysis
This section evaluates **employee satisfaction levels**, helping HR understand whether **low satisfaction contributes to attrition**.

### 🔹 Employee Distribution by Age Group
Displays how employees are distributed across different **age groups**, helping organizations understand workforce demographics.
